그렇게 바로 시작했습니다.

참고한 내용

react-notion-x 레포지터리에 있는 example 들 중, full 내용을 가장 많이 참고해서 가져왔습니다. 역시 누가 짜둔 코드가 최고였습니다.
 
react-notion-x/examples/full at master · NotionX/react-notion-x
This is a Next.js project bootstrapped with create-next-app . This is a more full-featured Next.js example project using react-notion-x, with the most important code in pages/[pageId].tsx and components/NotionPage.tsx . You can view this example live on Vercel.
https://github.com/NotionX/react-notion-x/tree/master/examples/full
 
하지만, 몇가지 문제가 있었습니다.
 
  1. root 도메인이 아니라, subdomain에 호스팅해보려 했습니다.
    1. 즉. singun11.wtf이 아닌, singun11.wtf/notion-blog 로 해보려했었습니다.
    2. 이 때문에, 계속 루트 도메인에 pageID가 붙어 다른 페이지로 이동되는 이슈가 있었습니다.
    3. sub domain에서 활용해봤으니 이제 blog.singun11.wtf 으로 바꿔서 해볼 예정입니다.
  1. 한글 title이 있는 경우, 제대로 링크가 생성되지 않는 이슈
